To set up locally for testing:

	1) Download as ZIP from the GitHub repo
	2) Extract into your Apache document root and make sure the folder name is "UPresent"
	3) Create a folder called "upload" in the UPresent directory
	4) Run this command in terminal after you cd into the UPresent directory: > sudo chmod 0777 -R upload
	5) Run the SQL file found in sql/create&populatedb.sql to create the databases
